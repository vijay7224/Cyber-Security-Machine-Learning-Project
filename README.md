🔐 AI-Based Intrusion Detection System (IDS)
📌 Overview

This project is an AI-powered Intrusion Detection System built using Machine Learning on the NSL-KDD dataset. It analyzes network traffic features to detect whether the traffic is normal or an attack.

🚀 Features

Detects network intrusions using AI/ML
Uses 41 network traffic features
High accuracy (~99%) using Random Forest
Feature importance analysis included

🧠 Tech Stack
Python
Pandas, NumPy
Scikit-learn
Matplotlib

📊 Dataset
NSL-KDD Dataset
Contains normal and multiple attack types (DoS, Probe, R2L, U2R)

⚙️ Project Workflow

Data Loading
Data Preprocessing
Label Encoding
Train-Test Split
Model Training (Random Forest)
Feature Importance Analysis
Model Evaluation
Prediction

🤖 Model
Random Forest Classifier
Achieved ~99% accuracy
